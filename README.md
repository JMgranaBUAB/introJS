# introJS

Un proyecto educativo para introducirse en los conceptos fundamentales de JavaScript.

## Descripción

Este proyecto contiene ejemplos prácticos de los conceptos básicos de JavaScript, incluyendo:

- **Variables y constantes**: `const`, `let`
- **Tipos de datos**: números, strings, booleanos, arrays
- **Arrays multidimensionales**: manejo de estructuras de datos complejas
- **Operaciones matemáticas**: suma, resta, multiplicación, división
- **Funciones**: declaración y uso de funciones
- **Control de flujo**: bucles `for`, condicionales
- **Manejo de errores**: validación de entrada

## Estructura del Proyecto

```
introJS/
├── index.html          # Página HTML principal
├── js/
│   └── index.js        # Código JavaScript con ejemplos
└── README.md           # Este archivo
```

## Características

- Ejemplos de declaración de variables y constantes
- Operaciones con arrays
- Funciones matemáticas (suma, resta, multiplicación, división)
- Validación de entrada (división por cero)
- Uso de bucles para iterar sobre arrays

## Uso

1. Abre el archivo `index.html` en tu navegador
2. Abre la consola del navegador (F12) para ver los resultados
3. Revisa el archivo `js/index.js` para entender el código
